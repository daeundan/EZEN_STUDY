DOM 응용 및 활용!

-DOM안에 요소를 생성 // 제거 // 제어!!!
-html문서 안에 있는 "모든 요소 = 태그 / 텍스트 / 속성 / 속성값"를 객체의 형태로 만들어서
자바 스크립트 문서에서 보다 효율적으로 확인할 수 있도록 하는것 !

(1) 문서 안에 새로운 Node를 생성하고자 할때
> createElement("p")

(2) 특정 태그 형식의 Node안에 입력될 텍스트를 생성하고자 할 때
> createTextNode("입력하고 싶은 텍스트")

(3) 특정 노드 요소를 사우이 노드에 자식 요소로 삽입하고자 할 때
> 상위 노드명.appendChild("자식요소 노드")

(4) 특정 노드에 속성을 생성하고 적용하고자 할 때
> 속성 생성 : img(src)필요한것 처럼! createAttribut("속성명")
> 속성값 적용 : 노드명.value = 속성값

(5) 기본적인 상품정보 json 생성 >
> js 데이터 = > DOM 출력!
> 데이터 재활용 - Html을 가급적 조금만 사용할 것
> js를 활용해서 최초로 사용한 데이터를 재활용 할 수 있음
> 신규 데이터 생성

< 나중에 만들 다이어리 앱>
특정 월에 입력한 일기만 필터링하여 가져올 수 있음
 * 주의해야 될 점. 컴퓨터만 인식하는 0, 1이라는 값이 있기 때문에 컴퓨터가 날짜를 읽는 숫자가 다름
(ex)
7/1 ~ 7/31
8/0 ~ 8/1 

(6) 현 시점의 날짜 정보 데이터를 생성해주는 함수 >
* prototype 함수 *
> 년, 월, 일, 시, 분, 초 모든 정보를 알려줌
(ex) const [이름] = new (예약어) Date(무조건 대문자로 시작)

(7) 특정 노드 중간에 삽입하고자 할 때 >
insertBefore(새로운 노드 : 1번째 인자값, 기존 노드 : 2번째 인자값)

(8) 특정 노드를 삭제하고자 할 때 >
> 노드.remove()

(9) 부모 노드 요소를 찾을 때 >
> 노드.parentNode

(10) 부모 요소를 기준으로 자식 노드 요소를 찾아서 제거하고자 할 때 >
> 부모노드.removeChild(자식노드) - 부모 노드요소 9번보다 이 조합을 더 많이 씀
